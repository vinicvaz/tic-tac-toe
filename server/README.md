 # Tic Tac Toe - Server
 
 # Install dependencies
 The project uses node v15
 
 To install the dependencies run:
 ```
 yarn install
 ```
 
 # Run the server
 First create a .env file following the .env.example file  
 - Run in development mode
 ```
 yarn dev
 ```
 - Running in production mode
 ```
 yarn start
 ```
